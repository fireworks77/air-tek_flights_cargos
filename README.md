# Project for Air-Tek flights schedule and order cargo list## This project is to generate flight schedule and load orders## This project has 3 classes- `JSON`- `Utilities`- `Flights_Cargos_Schedule`## How to run- Import the entire project to Eclipse or any Java IDE- Click `Flights_Cargos_Schedule` class and `Run AS Java Application`## Runnable jar- Open a terminal/command window- Run command `Java -jar flights_cargos.jar`
